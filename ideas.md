# SCL-90 心理自评量表 - 设计方案探索

## 项目背景
SCL-90 是一个专业的心理健康筛查工具，需要传达专业性、可信度和人文关怀。用户在使用时处于相对敏感的心理状态，设计应该让人感到安全、被理解和被尊重。

---

## 方案一：现代医学风格 (Modern Medical Aesthetic)
**设计运动**：当代医疗设计 + 极简主义

**核心原则**：
- 清晰的信息层级，减少认知负担
- 冷静、专业的视觉语言
- 柔和的过渡，避免突兀感
- 数据可视化与人文关怀的平衡

**色彩哲学**：
- 主色：靛蓝 (#6366f1) - 传达专业性和信任
- 辅色：柔和的灰蓝 (#e0e7ff) - 背景，营造宁静感
- 强调色：温暖的琥珀 (#f59e0b) - 用于重要提示
- 危险色：柔和的红 (#ef4444) - 警告信息
- 背景：极浅灰 (#f8fafc) - 减少视觉疲劳

**布局范式**：
- 卡片式设计，每个问题独立呈现
- 垂直流式布局，引导用户逐步完成
- 结果页面采用仪表板风格，多维度展示

**标志性元素**：
1. 柔和的圆角卡片（border-radius: 16px）
2. 微妙的阴影和边框，营造深度感
3. 渐进式的视觉反馈（选中状态的颜色变化）

**交互哲学**：
- 即时反馈：选择选项时立即显示视觉变化
- 平滑过渡：所有状态变化都有 0.2-0.3s 的过渡
- 浮动操作栏：固定在底部，随时可见

**动画指南**：
- 卡片进入：淡入 + 轻微上升（200ms）
- 选项选中：背景色平滑变化 + 边框高亮
- 结果展示：数字逐个加载，从 0 到最终值

**排版系统**：
- 标题：PingFang SC / Microsoft YaHei，700 weight，1.8rem
- 正文：系统字体栈，400 weight，0.95rem
- 小文本：0.85rem，用于说明和引用

## 方案二：温暖人文风格 (Warm Humanistic Design)
**设计运动**：新人文主义 + 温暖极简

**核心原则**：
- 强调人的情感和体验
- 温暖而非冷漠的专业感
- 视觉上的拥抱感，让用户感到被支持
- 渐进式的引导，减少心理压力

**色彩哲学**：
- 主色：温暖的紫罗兰 (#8b5cf6) - 传达同理心和支持
- 辅色：柔和的玫瑰 (#fce7f3) - 温暖的背景
- 强调色：清爽的绿 (#10b981) - 积极和希望
- 危险色：柔和的橙红 (#f97316) - 友好的警告
- 背景：奶油色 (#fffbf0) - 温暖感

**布局范式**：
- 非对称布局，左侧问题、右侧进度
- 渐进式展开，一次展示 3-5 个问题
- 结果页面采用故事化展示

**标志性元素**：
1. 柔和的渐变背景
2. 手绘风格的插图和装饰
3. 温暖的色彩过渡

**交互哲学**：
- 鼓励性反馈：每个选择都有正面的确认
- 进度可视化：清晰显示完成进度
- 同理心提示：在关键点提供人文关怀的信息

**动画指南**：
- 页面进入：从下往上的温暖淡入
- 问题展开：柔和的展开动画，带有轻微的旋转
- 结果呈现：逐个显示维度，带有庆祝感的动画

**排版系统**：
- 标题：Poppins / 思源黑体，600 weight，1.8rem
- 正文：系统字体栈，400 weight，0.95rem
- 强调文本：使用颜色而非加粗

## 方案三：极简科学风格 (Minimal Scientific Design)
**设计运动**：数据可视化极简主义 + 学术风格

**核心原则**：
- 数据驱动的视觉设计
- 最小化装饰，最大化信息传达
- 科学严谨性与美学的结合
- 清晰的视觉层级和对比

**色彩哲学**：
- 主色：深蓝 (#1e40af) - 学术和科学感
- 辅色：浅灰 (#f3f4f6) - 中立背景
- 强调色：深绿 (#065f46) - 正面结果
- 危险色：深红 (#991b1b) - 需要关注
- 背景：纯白 (#ffffff) - 科学感

**布局范式**：
- 网格布局，问题以表格形式呈现
- 结果页面采用科学报告风格
- 维度分析采用平行坐标或雷达图

**标志性元素**：
1. 精确的几何形状和网格
2. 单色或双色的数据可视化
3. 学术风格的排版

**交互哲学**：
- 高效率的交互，减少点击次数
- 清晰的数据展示，支持导出
- 学术引用和参考文献的展示

**动画指南**：
- 最小化动画，保持专业感
- 数据变化时的平滑过渡
- 结果加载时的进度指示

**排版系统**：
- 标题：Georgia / 思源黑体，700 weight，1.8rem
- 正文：Courier New / 等宽字体，400 weight，0.9rem
- 数据：等宽字体，用于数字和统计

---

## 最终选择：**方案一 - 现代医学风格**

我选择了**现代医学风格**作为设计基础，原因如下：

1. **专业性与信任**：靛蓝色传达医学专业性，符合 SCL-90 的学术背景
2. **用户心理**：在进行心理评估时，用户需要感到被专业对待，同时不过于冷漠
3. **可访问性**：清晰的信息层级和充足的空白使所有用户都能轻松使用
4. **扩展性**：卡片式设计易于维护和扩展，适合长期迭代
5. **平衡感**：结合了医学的严谨性和人文的温暖性

### 设计决策总结

| 方面 | 决策 |
|------|------|
| 主色系 | 靛蓝 (#6366f1) + 灰蓝辅色 |
| 背景 | 极浅灰 (#f8fafc)，减少视觉疲劳 |
| 排版 | 系统字体栈 + 中文字体，清晰易读 |
| 布局 | 卡片式垂直流，引导用户逐步完成 |
| 交互 | 即时反馈 + 平滑过渡 |
| 动画 | 200-300ms 的淡入和颜色变化 |
| 结果展示 | 仪表板风格，多维度数据可视化 |

这个设计既保持了原始 HTML 版本的专业感，又通过现代化的 React 组件和 Tailwind CSS 提升了用户体验。
